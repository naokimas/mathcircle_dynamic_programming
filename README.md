# Dynamic programming

Here are materials for my [Michigan Math Circle](https://sites.lsa.umich.edu/mathcircle/) lecture for middle and high students on dynamic programming, on 2/19/2026.

## Lecture notes

* `worksheet-dp-math-circle.pdf`: Main lecture notes. I distributed a copy of this to each student. In 90 minutes, I was unable to go into section 4.

* `supp-lecture-notes-dp-math-circle.pdf`: Supplementary lecture notes. This one has answers to the questions and what I wanted to speak or write on the blackboard in class. Not to be distributed to students.

## Codes to generate figures

I generated drafts of these codes by AI and then adjusted it.

* `dp_maze_*.py`: Generate the figures in section 1.
* `frog_stones_{1,2,3}.py`: Generate the figures in section 2.
* `candy_collection_{1,2,3}.py`: Generate the figures in section 3.
